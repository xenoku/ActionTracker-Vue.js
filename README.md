# ActionTracker - Vue.js

Персональный трекер занятости

## Автор проекта

Король Максимилиан Игоревич, гр. 609-91

## Описание проекта

Приложение позволяет аутентифицированному пользователю создавать виды деятельности и в любой момент переключаться между ними. Приложение запоминает распределение занятости по времени и подготавливает по нему инфографику. Перевод в SPA на Vue.js
